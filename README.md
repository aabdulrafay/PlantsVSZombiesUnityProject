# Plants vs Zombies – Unity Game Project

## Overview
This project is a 2D Plants vs Zombies inspired game developed in Unity as part of the Game Programming course.


## Features

### 🌱 Plants
- Peashooter
- Sunflower
- WallNut

### 🧟 Zombies
- Normal Zombie
- Conehead Zombie
- Buckethead Zombie
- Flag Zombie

### 🎮 Gameplay Features
- Grid-based plant placement
- Mouse controls
- Animator State Machines
- Collision detection
- Projectile attacks
- HP/Damage system
- Start/Pause/Quit UI system

## Technologies Used
- Unity 2020.3.22f1
